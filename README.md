## 🟢 Overview
The **SSF Moisture Optimizer** is a specialized computational tool designed for researchers working with filamentous fungi and agro-industrial waste. It standardizes complex laboratory calculations into a high-performance, **Emerald Dark** themed web interface.

Developed by **Adeoye Joshua Oluwafemi (CodeByPappy)**, this suite bridges the gap between microbiological wet-lab research and modern software engineering.

## 🛠️ Features
The suite consists of three core modules:
1. **Moisture Content Analyzer:** Instantly determine substrate moisture levels using dry-basis equations.
2. **Moisture Prep Optimizer:** Calculate exact water and inoculum volumes needed to reach target moisture levels for fungal growth.
3. **Yield Analysis:** Automate the calculation of bioconversion efficiency and metabolic productivity metrics.

## 🧪 Scientific Methodology
The tool utilizes standardized bioprocess engineering formulas:
- **Moisture Content:** $\frac{Wet\ Mass - Dry\ Mass}{Wet\ Mass} \times 100$
- **Target Hydration:** Precision water activity ($a_w$) adjustment based on substrate dry weight.

## 💻 Tech Stack
- **Architecture:** Single-file "Offline-First" design for laboratory reliability.
- **Languages:** HTML5, CSS3 (Emerald Dark Theme), and Vanilla JavaScript.
- **Performance:** Zero external dependencies for instant loading in low-connectivity areas.
